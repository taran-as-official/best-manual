Регистрация в GitHub

  • Переходим на официальный сайт GitHub, сразу в раздел {% extlink создания аккаунта https://github.com/join?ref_cta=Sign+up&ref_loc=header+logged+out&ref_page=%2F&source=header-home %}
  • Заполняем поля в форме, проходим верификацию и нажимаем Create account
  • Выберите ответы на вопросы
  • Укажите интересующие языки программирования, фреймворки, отрасли
  • Жмем кнопку Complete setup
  • Зайдите в почту, которую указывали при регистрации и подтвердите аккаунт

Создание репозитория

  • Авторизовываемся на GitHub и в верхнем правом углу нажимаем на +->New repository
  • Заполняем Repository name, по желанию Description (описание) и нажимаем Create repository

Установка Git

  • Переходим на официальный сайт и скачиваем {% extlink Git для Windows http://git-scm.com/download/win %}
  • Запускаем скачанный файл, жмем везде Next, обязательно убедитесь, что стоит галочка напротив Windows Explorer integration

Настройка репозитория Git

  • На компьютере выбираем папку, в которой будет хранится проект, переходим в неё нажимаем на пустом месте правой кнопкой мыши и выбираем Git Bash Here
  • Открываете профиль в GitHub, зарегистрированный ранее, и переходим в созданный репозиторий
  • Выбираете из списка нужный репозиторий
  • Копируем полный путь репозитория, он может быть сразу на странице (рис. 1), либо жмем Clone or download и также копируем полный путь репозитория (рис. 2)

    рис. 1

    рис. 2

  • Возвращаемся в окно Git Bash, открытое в первом пункте для выгрузки репозитория на свой ПК. Пишем команду согласно шаблона

    Шаблон:

    git clone вставляем_строку_скопированную_в_пункте_выше

    Пример кода:

    git clone https://github.com/taran-as-official/best-manual.git

    Закройте окно Bash

  • На ПК в папке, откуда запускали команду Git Bash Here, должна появиться папка с названием вашего репозитория
  • Имитируем изменения в проекте. Зайдите в папку репозитория, создайте, текстовый файл, назовите "test", откройте его напишите "test" и сохраните, на пустом месте нажмите правой кнопкой мыши и нажмите Git Bash Here
  • В окне Git Bash зададим глобальные настройки, выполните две команды по примеру ниже, жирным шрифтом выделены части, в которых укажите свой email и логин, который указывали при регистрации в GitHub (ковычки не удаляем!)

    Пример кода:

    git config --global user.email "best.manual.info@gmail.com"
    git config --global user.name "taran-as-official"

  • Загрузим изменения на сервер GitHub. Продолжаем вводить по очереди команды ниже

    git add .
    git commit -m 'my first commit'
    git push

    При вводе последней команды нужно будет ввести логин и пароль.
    ВАЖНО: при вводе пароля символы не будут отображаться, но они по факту вводятся

    РЕКОМЕНДАЦИЯ: отклоните авторизацию через логин и пароль, появится окно авторизации через SSH, вот через SSH и авторизуйтесь

  • Зайдите на сервер GitHub, обновите страницу репозитория, добавленный файл должен отобразиться на сервере!

Если в ходе выполнения мануала вы выявите ошибки, либо пункты не будут соответствовать действительности напишите об этом в комментариях, на почту, либо соцсети
