Сейчас вы узнаете как быстро настроить доступ в Github по SSH ключу. Больше не придется вводить логин и пароль при работе с Git.

  • В папке с проектом нажимаем правой кнопкой мыши по пустому месту и нажмаем Git GUI Here
  • Далее Help->Show SSH Key
  • После нажатия Generate key вам будет предложено ввести контрольную фразу - оставьте значение пустым, иначе придется каждый час вводить эту фразу
  • Сгенерированный ключ показан ниже, теперь копируем его, нажав Copy To Clipboard
  • Логинимся на сайте {% extlink Github https://github.com/ %} и открываем раздел Settings->SSH and GPG keys
  • Жмем New SSH key
  • В поле "Title" вводим название ключа (для каждого ПК и ОС свой SSH), в поле "Key" вставляем скопированный ранее SSH ключ. В конце нажимаем Add SSH key

Готово!

Теперь можете перейти в Git, ввести команду git push, будет задан вопрос уверены ли мы в этом подключении напишите "yes" и нажмите Enter

Спасибо за внимание!
