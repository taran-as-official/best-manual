Включение AutoReplace

Запускаем SQL Developer, переходим Tools->Preferences...

В появившемся окне переходим Database->SQL Editor Code Templates, ставим галочку напротив Autoreplace и нажимаем OK

Проверим работу, откройте окно для ввода кода, напишите cf и нажмите пробел, должен появится шаблон как на картинке ниже

Настройка AutoReplace

Вернитесь обратно в настройки, где включали функцию Autoreplace, чтобы добавить свой шаблон нажмите Add Template, для удаления Remove Template, для изменения текущих шаблонов просто нажимаете на них и меняете

Далее пойдут ряд шаблонов, которые 100% пригодятся в ходе написания кода

Id Template
se select * from
wh where
de delete
hex hextoraw('')
raw rawtohex('')

Рекомендую таблицы, которые часто используете, также занести в шаблон
